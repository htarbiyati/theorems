The single layer potential in three dimensions

Let \(\eta\colon \Gamma \rightarrow \mathbb{R}\) be a given function defined on the boundary \(\Gamma\). We then define the single layer potential \(S_\eta \colon \mathbb{R}^3\setminus \Gamma \rightarrow \mathbb{R}\) with the formula:

\[(S\eta)(z)=\int_{\Gamma} \frac{\eta(y)}{4\pi|z-y|}\,d\Gamma(y)\]

the input function \(\eta\) will be called a density. If \(\eta \in L^{1}(\Gamma)\) and \(u=S\eta\) then \(u \in C^\infty(\mathbb{R}\setminus \Gamma)\), and

\[\Delta u=0 \hspace{3mm} \mbox{in} \hspace{3mm} \mathbb{R}^3\setminus\Gamma\]

\[u(z)=a\frac{1}{|z|} + \mathcal{O(|z|^{-2})} \hspace{3mm} \mbox{as}\,\, |z| \rightarrow \infty\]

where \(a=\frac{1}{4\pi}\int_{\Gamma}\eta\,d\Gamma\).

Some numerical methods

Consider a partition of \(\Gamma\) in relatively open disjoint patches \(\{T_1,\dots,T_N\}\) \[ T_i \cap T_j=\emptyset\,,\,i\ne j,\hspace{6mm}\Gamma=\cup_{j=1}^{N}\overline{T_j} \] and the set of piecewise constant functions with respect to this partition \[ X_h=\left\{\eta^h:\Gamma\rightarrow\mathbb{R}\colon\eta^h\mid T_j\in\mathcal{P}(T_j)\,\, \forall j \right\} \]

where \(\mathcal{P}(\Omega)\) is the set of constant functions on the domain \(\Omega\).(We will use a script \(h\) to denote discretization.The meaning of this index will be clear when we study approximation properties of the space \(X_h\) in different norms. In that case \(h\) will be the maximum diameter of the elements of the partition.) A basis for \(X_h\) is easy to define using the characteristic functions of the elements of the partition: \[ \chi_j= \begin{cases} 1 & \mbox{in} \,\, T_j\,,\\ 0 & \mbox{elsewhere.} \end{cases} \]

Galerkin method

For an abitrary density \(\eta\) and for points \(x \in \Gamma\), we define \[ (V\eta)(x)=\int_{\Gamma}\frac{\eta(y)}{4\pi|x-y|}\,d\Gamma(y) \] We look for \(\eta^h \in X_h\) such that \[ \int_{\Gamma}\mu^h(x)(V\eta^h)(x)\,d\Gamma(x)=\int_{\Gamma}\mu^h(x)\beta_{0}(x)\,d\Gamma(x)\hspace{6mm} \forall\mu^h\in X_h \] for given boundary data \(\beta_0\colon \Gamma \rightarrow \mathbb{R}\) It is clear that any solution of this equation is also a solution of \[ \int_{T_i}(V\eta^h)(x)\,d\Gamma(x)=\int_{T_i}\beta_0(x)\,d\Gamma(x) \hspace{6mm} i=1,\dots,N \] since we only need to restrict testing in previous equation to the basis functions \(\chi_i\). However, linearity of integration and the fact that \(\{\chi_1,\dots,\chi_N\}\) is a basis for \(X_h\) shows that above equation also implies previous equation. We can go one step further by decomposing the unknown in the given basis. We thus look for \[ (\eta_1,\dots,\eta_N) \in \mathbb{R}^N, \hspace{6mm} \mbox{representing} \hspace{6mm} \eta^h=\sum_{j=1}^{N}\eta_j\chi_j\,, \] such that \[ \sum_{j=1}^{N}\left(\int_{T_i}(V(\chi_j)(x)\,d\Gamma(x)\right)\eta_j=\int_{T_i}\beta_0(x)\,d\Gamma(x)\hspace{6mm} i=1,\dots,N \] It clear from above equation that we are in the presence of an \(N \times N\) system of linear equations.

Further information in Chapter 2 of Yousef Saad book.
